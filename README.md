# Credify JS Example (React.js)

This project is an example project of Credify JS for React.js. This application is using a [backend example project](https://github.com/credify-pte-ltd/Nodejs-Credify-JS-Example).

## How to use

*NOTE: This project is using [Nodejs Example Project](https://github.com/credify-pte-ltd/Nodejs-Credify-JS-Example), so please clone this Nodejs project as well.*

```shell script
# Install dependencies
$ yarn
# Run a project
$ yarn start
```


The port needs to be 3000.

Hardcoded app config (`App.tsx`) is already registered as an OpenID Connect client in Credify sandbox system.
